# Autonomous Health Assistance and Medication Delivery System

## Project Description
This project is an IoT-based healthcare system designed to monitor patient health conditions and deliver medication automatically. The system helps improve patient care by providing real-time health monitoring and automated medicine delivery.

## Features
- Real-time health monitoring
- Automatic medication delivery
- Sensor-based patient monitoring
- LCD display for health status
- Alert system for abnormal health conditions

## Technologies Used
- Arduino
- IoT Sensors
- Embedded Systems
- Motor Driver
- LCD Display

## Project Objective
The objective of this project is to assist patients by continuously monitoring health parameters and ensuring timely delivery of medication.
